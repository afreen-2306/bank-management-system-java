# Student Result Management System
A console-based Java application that calculates the total marks, average, grade, and result of a student.

## Features 
- Accepts student name and roll number
- Takes marks for three subjects
- Calculates total and average
- Assigns grade based on performance
- Displays pass or fail result
- Validates marks input(0-100)

  - Methods
