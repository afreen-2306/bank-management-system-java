# Bank-management-system-java
## Description
This is a console-based Java application that simulates basic banking operations.
It allows users to create an account, deposit money, withdraw money, and check balance.
The project was developed as part of learning core Java concepts.

##Features
- Account creation
- Deposit money
- Withdraw money
- Balance enquiry
- Menu-driven console interface

## Technologies Used
- Java
- OOP concepts (Classes, objects, Methods)
- Scanner for user input

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (VS Code/ Eclipse/ IntelliJ)
3. Compile and run the 'bank.java' file
